기술 스택 / 사용 가능 기술

HTML

  • 시맨틱 태그(header, nav, main, section, footer)를 통해 명확한 문서 구조를 설계할 수 있다.
  • 이미지의 alt 속성, 링크의 title, 텍스트의 em/strong 등 접근성을 고려한 마크업을 사용할 수 있다.

CSS

  • 테두리(border),배경(background), 글자 스타일(margin, gap)과 border, background, box-shadow를 적절히 활용할 수 있다.
  • 디자인 토큰(색상, 반경, 그림자)을 사용해 일관성 ㅁ는 스타일을 구성할 수 있다.
  • display: grid와 flex를 혼합해 반응형 레이아웃을 만들 수 있다.
